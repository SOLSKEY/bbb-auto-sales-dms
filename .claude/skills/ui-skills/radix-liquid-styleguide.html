Radix Themes + Liquid Glass

A premium dark-mode design system with liquid glass morphism surfaces

Design System

Core Concept

The Liquid Container system applies a signature glass morphism effect to all UI surfaces using layered inset shadows. This creates a soft, beveled appearance with inner color glow.

Apply to:

  • ✓ Buttons, Cards, Badges
  • ✓ Inputs, Selects, Dropdowns
  • ✓ Checkboxes, Radios, Switches
  • ✓ Tabs, Callouts, Tables
  • ✓ Any element with a background or border

Exclude:

  • ✗ Standalone text (stays normal)
  • ✗ Icons without container backgrounds
  • ✗ Raw data values

Implementation

Core Liquid Shadow Formula

.liquid-surface {
  box-shadow:
    /* Outer shadows for depth */
    0 0 6px rgba(0, 0, 0, 0.03),
    0 2px 6px rgba(0, 0, 0, 0.08),
    /* Inner bevel highlights */
    inset 3px 3px 0.5px -3.5px rgba(R, G, B, 0.15),
    inset -3px -3px 0.5px -3.5px rgba(R, G, B, 0.85),
    /* Fine edge highlights */
    inset 1px 1px 1px -0.5px rgba(R, G, B, 0.6),
    inset -1px -1px 1px -0.5px rgba(R, G, B, 0.6),
    /* Inner color glow */
    inset 0 0 6px 6px rgba(R, G, B, 0.12),
    inset 0 0 2px 2px rgba(R, G, B, 0.06),
    /* Outer ambient shadow */
    0 0 12px rgba(0, 0, 0, 0.15);
  border: 1px solid rgba(R, G, B, 0.4);
  background-color: rgba(0, 0, 0, 0.2);
}

Apply Colors via CSS Variables

/* Define color via CSS custom properties */
.liquid-cyan {
  --liquid-r: 6;
  --liquid-g: 182;
  --liquid-b: 212;
}

/* Or via data attribute */
[data-liquid-color="cyan"] {
  --liquid-r: 6;
  --liquid-g: 182;
  --liquid-b: 212;
}
